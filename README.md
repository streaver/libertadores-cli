# Usage
<!-- usage -->
```sh-session
$ npm install -g libertadores-cli
$ libertadores COMMAND
running command...
$ libertadores (-v|--version|version)
libertadores-cli/0.0.0 darwin-x64 node-v11.13.0
$ libertadores --help [COMMAND]
USAGE
  $ libertadores COMMAND
...
```
